Web AI Summit 2025: Bangle.js Demo

Press button to enable gesture recognition!

By clicking this button we'll request to connect to your Bangle.js watch via Bluetooth, after which we'll stream sensor data in real-time, to classify it entirely locally (no cloud) using a Web AI model that detects when you clap your hands for a few seconds!

This is just a simple demo. We encourage you to hack after the show with all the other sensor data available, and let us know what you make! View source code on our GitHub repo..

Note: iOS users will need to install a Web Bluetooth capable browser such as Bluefy or WebBLE. Laptop users please use Chrome.
