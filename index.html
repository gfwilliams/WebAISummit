Web AI Summit 2025: Bangle.js Demo

Press button to enable gesture recognition!

Upon clicking this button we'll request to connect to your Bangle.js watch via Bluetooth, after which we will stream sensor data from the watch in real-time, to then classify the data entirely locally (no cloud) using a Web AI model that should detect when you clap your hands!

This is just a simple demo, we encourage you to hack after the show with all the other sensor data you can grab from the watch, and let us know what you make! You can view the source code of this webpage on our GitHub repo. You can learn more about Web AI with our 15 min videos.

Note: iOS users will need to install a Web Bluetooth capable browser such as Bluefy or WebBLE. Laptop users please use Chrome.
